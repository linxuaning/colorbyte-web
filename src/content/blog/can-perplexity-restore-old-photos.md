# Can Perplexity AI Restore Old Photos?

Perplexity AI is an AI-powered search engine — it answers questions using web sources and provides cited responses. If you're wondering whether it can restore old family photographs:

**Perplexity cannot restore old photographs.** It's an AI search tool that finds and synthesizes information from the web. It does not process, edit, or modify images.

---

## What Perplexity Can Do

**Research restoration options:** Ask Perplexity "what's the best tool for restoring old photos" and it will search the web and compile a cited answer.

**Compare tools:** Perplexity can pull reviews and comparisons from web sources about photo restoration services.

**Explain techniques:** Perplexity can explain how AI photo restoration works, what CodeFormer does, and other technical questions.

**What it cannot do:** Upload a photo and get back a restored version. Perplexity has no image editing or processing capabilities.
