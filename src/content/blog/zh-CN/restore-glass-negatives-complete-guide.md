## 认识玻璃底片

玻璃底片在不同年代以多种规格存在,每一类都需要相应的修复方法。

### 玻璃底片的种类

**湿版火棉胶底片（1851—1880 年代）**

最早实现商业化成功的负片工艺：
- 玻璃涂布火棉胶（火棉溶于乙醚）
- 曝光前即时增感
- 趁湿冲洗显影
- 影像极其锐利
- 乳剂层异常脆弱
- 常带有特征性的瑕疵和不规则痕迹

**干版明胶底片（1871—1920 年代）**

允许预涂的改良工艺：
- 玻璃上涂布明胶乳剂
- 可在使用前长期存放
- 比湿版方便
- 直至 1900 年代初一直是专业影楼的标准
- 乳剂层比湿版更稳定
- 通常保存得更好

**幻灯片（1850—1950 年代）**

供放映使用的玻璃正片：
- 由底片接触印相而来
- 用于讲座、教育和娱乐
- 常以手工上色
- 通常带有保护盖玻璃
- 是珍贵的历史文献

如需了解更全面的历史照片修复技法,请参阅我们的[老照片修复](/vintage-photo-restoration)指南。

### 玻璃底片常见的损伤

**物理损伤**

100 多年后玻璃极易破裂：
- 板面开裂（撞击或温差应力造成）
- 边角破损（搬运或存放造成）
- 完全碎裂（多块碎片）
- 缺口与边缘磕损

**乳剂层劣化**

感光涂层不断老化：
- 乳剂层剥离（从玻璃上脱落）
- 起片与脱落
- 银化与镜面反光
- 褪色及密度降低
- 化学污染

**环境损伤**

存储条件影响保存效果：
- 真菌生长（乳剂层霉变）
- 水渍损伤（受潮、浸泡）
- 与纸质封套粘连
- 灰尘碎屑堆积
- 氧化与变色

## 关键安全与操作准则

玻璃底片既物理脆弱,也可能存在潜在危害。

### 个人安全

**健康危害**

旧玻璃底片可能含有危险物质：
- **碎玻璃**：锋利的边缘和碎屑
- **真菌孢子**：来自霉菌的呼吸道危害
- **化学残留**：银化合物、显影液、定影液
- **火棉胶乳剂**：可能含有有害化合物

**防护装备**

务必做好防护：
- 丁腈手套（切勿赤手操作）
- 若有霉菌则佩戴 N95 口罩
- 处理破损玻璃时佩戴护目镜
- 工作面铺设保护层（一次性覆盖物）
- 急救用品就近备齐

### 安全搬动玻璃底片

**基本搬动规则**

1. 始终只捏底片边缘
2. 自下方完整托起整片
3. 切勿无隔层堆叠玻璃片
4. 在垫衬过的桌面上操作
5. 一次只移动一片
6. 竖立存放,绝不平摞

**针对破损的玻璃片**

特别留意,避免再次损坏：
- 托住所有碎片
- 在底下垫一块硬板
- 让碎片保留原始相对位置
- 切勿尝试"拼接复原"
- 用照片记录碎片位置
- 价值贵重的玻璃片建议交由专业修复

**针对乳剂层翘起的玻璃片**

脆弱的乳剂层需要极为谨慎：
- 尽量减少搬动
- 始终保持水平（绝不竖立）
- 切勿触碰乳剂面
- 务必尽快寻求专业修复
- 在进一步损失前立即数字化

## 玻璃底片的数字化

合格的数字化能在物理劣化加剧之前先把影像保留下来。

### 扫描玻璃底片

**扫描仪的选择**

带透射稿片附件的平板扫描仪很合适：
- 爱普生 Perfection 系列（V600、V850）
- 专用胶片扫描仪（前提是底片放得下）
- 支持高分辨率（光学 4800 DPI 以上）
- 必须具备透射光源

**扫描流程**

1. **彻底清洁扫描台面**
2. **将底片乳剂面朝下**置于扫描台（乳剂面背离玻璃台面）
3. **使用片夹或定位工具**避免移位
4. **以正片模式扫描**（多数软件都有反相选项）
5. **高分辨率**：2400—4800 DPI 以保留最大细节
6. **48 位色深**(即便是黑白底片也有助于保留细腻层次)
7. **关闭自动校正**
8. **保存为无压缩 TIFF**

**为什么必须乳剂面朝下**

把乳剂面朝向扫描玻璃会导致：
- 光线穿过玻璃厚度产生散射
- 锐度下降
- 损失精细细节
- 始终让乳剂面朝下（背离扫描玻璃）

### 翻拍玻璃底片（更推荐的方法）

翻拍往往比扫描效果更好：

**器材配置**

- 具备微距能力的数码相机
- 稳定的三脚架
- 灯箱或背光设置
- 翻拍架或顶置安装方案
- 快门遥控

**布光方法**

**方案一：灯箱**
1. 将玻璃底片放在灯箱上
2. 相机俯拍架设
3. 测光以保证均匀曝光
4. 以 RAW 格式拍摄

**方案二：窗口光**
1. 将底片贴近窗户（避免直射光）
2. 相机从室内一侧对准底片
3. 均匀、漫射的日光最为理想
4. 仔细测光,避免过曝

**方案三：LED 平板背光**
1. 以 LED 平板做背光源
2. 在面板上覆盖一层柔光材料
3. 把底片放在柔光面前方
4. 相机从正面拍摄

**相机设置**

- 必须使用 RAW
- 感光度：最低（100—200）
- 光圈：f/8 至 f/11（最佳锐度区间）
- 对乳剂颗粒手动对焦
- 曝光：以高光为基准（避免溢出）
- 包围曝光

### 拍摄/扫描后的反相处理

玻璃负片必须转换为正像：

**Photoshop / Affinity Photo 中的操作**：
1. 打开扫描或拍摄到的负片图像
2. 图像 > 调整 > 反相（Cmd/Ctrl+I）
3. 此时即显示为正片
4. 进入修复流程

**用曲线替代反相**：
1. 新建一个曲线调整图层
2. 翻转曲线（把右下端拖到左上,左上端拖到右下）
3. 比简单反相更可控
4. 可同时进行调整

## 数字化后修复玻璃底片影像

完成数字化与反相后,就可以用一系列技法恢复画质。如果想把裂缝修补、细节复原与最终增强一气呵成,使用 AI 驱动的[照片修复工具](/old-photo-restoration)能在保留历史准确性的前提下显著加速整个流程。

### 修复裂痕与断裂

**数字化裂痕去除**

反相之后的图像中,裂痕呈现为深色线条：

1. 仿制图章不透明度设为 50%
2. 从紧邻裂痕的区域取样
3. 沿裂痕路径仿制
4. 保持 100% 缩放进行操作
5. 频繁更换取样点
6. 一层层叠加

**针对碎裂的玻璃片**

多块碎片需要特别处理：

1. 分别扫描或拍摄每一块碎片
2. 将所有碎片导入图像编辑软件
3. 各自放在独立图层上
4. 精确对齐
5. 用仿制图章和修复工具融合接缝
6. 必要时进行几何变换以求严丝合缝

### 去除灰尘、划痕与污点

玻璃底片往往积攒了 100 多年的杂质：

**污点修复画笔**：
- 处理小污点既快又好
- 在 100% 缩放下操作
- 笔刷略大于瑕疵
- 单击,而非拖动

**针对较大瑕疵的仿制图章**：
- 比修复画笔更可控
- 从附近干净区域取样
- 匹配纹理与色调
- 低不透明度（30—50%）逐步叠加

**蒙尘与划痕滤镜**（需谨慎使用）：
- 滤镜 > 杂色 > 蒙尘与划痕
- 数值要很低（半径 1—2,阈值 2—4）
- 通过图层蒙版选择性应用
- 不要全图使用（会软化细节）

### 调整密度与反差

玻璃底片常见的问题：
- 密度偏低（薄底,曝光不足）
- 密度偏高（厚底,曝光过度）
- 反差偏低（影调平淡）
- 化学雾翳（整体灰蒙蒙）

**用曲线还原层次**：

1. 新建曲线调整图层
2. 设定黑场（最暗的阴影区域）
3. 设定白场（最亮的高光）
4. 调整中间调以获得合适亮度
5. 做轻柔的 S 形曲线增加反差

**色阶替代方案**：

更简单的修正方式：
1. 新建色阶调整图层
2. 把黑场滑块拖到直方图最左侧有数据处
3. 把白场滑块拖到直方图最右侧有数据处
4. 调整中间调滑块控制亮度

### 处理乳剂层损伤

**银影褪色**

当银影像已经褪色：
- 大胆增加反差
- 用曲线扩展整个影调
- 可能需要牺牲部分高光或阴影细节
- 接受严重褪色底片本身的局限

**污渍与变色**

化学污渍呈现为不均匀的色块：
- 用减淡和加深工具均化色调
- 新建一个柔光叠加模式下的 50% 灰图层
- 用白色减淡,黑色加深
- 一层层叠加修正
- 保持整体的摄影质感

**乳剂缺失区域**

乳剂层完全脱落的位置：
- 先用内容识别填充（Photoshop）打底
- 用仿制图章从相似区域取样
- 根据上下文进行重建
- 重建结果要克制
- 记录哪些部分是补绘的

### 给玻璃底片扫描结果加锐

正确数字化的玻璃底片可以非常锐利：

**USM 锐化**：
- 数量：100—200%
- 半径：0.8—1.5 像素
- 阈值：2—5 级
- 较高数量是可以接受的（原片细节本就丰富）

**高反差保留锐化**：
1. 复制背景图层
2. 滤镜 > 其他 > 高反差保留（半径 2—4 像素）
3. 混合模式：叠加或柔光
4. 不透明度：50—80%

**选择性锐化**：
- 重点锐化关键细节（脸部、文字、主体元素）
- 背景少加锐
- 损伤或噪点严重的区域避免锐化

## 玻璃底片的物理保护

完成数字化后,妥善存放才能保护实物。

### 清洁玻璃底片（合适的时候）

**不该清洁的情况**：
- 乳剂层翘起或脱落
- 乳剂层严重损伤
- 出现霉菌（需专业修复介入）
- 极为珍贵的玻璃片（仅交专业人员处理）
- 状态不明、稳定性存疑

**安全清洁（仅限稳定的玻璃片）**：

1. 用软毛刷刷掉浮尘
2. 切勿用湿物接触乳剂面
3. 仅清洁玻璃面（板的背面）
4. 用棉签蘸蒸馏水
5. 极轻柔,不施加任何压力
6. 立即用洁净布擦干
7. 收纳前自然晾干彻底

### 合格的存放方案

**单独保护**

每块玻璃片都需要独立护套：
- 四面对折式无酸纸袋
- 档案级纸张（无木质素、含缓冲)
- 切勿使用透明纸（可能与乳剂粘连）
- 标签贴在外护套上,而不是底片本身

**存放方向**

玻璃板必须竖立存放：
- 切勿平摞（重量会损伤乳剂层）
- 在带凹槽的支架内竖立
- 不能让玻璃板倾斜
- 沿整条边缘均匀支撑

**存放盒**

专门设计的盒子能更好地保护玻璃底片：
- 档案级板材
- 每片之间有竖直隔片
- 尺寸合适（不会移位或挤压）
- 长期存放选用含缓冲的材料

**环境条件**

- 温度：65—70°F（18—21°C）
- 湿度：30—40%（保持稳定）
- 暗处存放（避光）
- 良好的空气流通
- 远离温湿度剧烈波动

## 幻灯片的特殊注意事项

幻灯片（玻璃正片）需要不同的处理思路。

### 结构与挑战

**幻灯片的构造**：
- 两块玻璃夹合
- 一块是带乳剂的影像板
- 另一块是保护性盖玻璃
- 边缘以胶带捆扎
- 通常贴有纸质标签

**劣化问题**：
- 牛顿环（玻璃接触产生的干涉花纹）
- 玻璃间封存的水汽
- 夹层内滋生霉菌
- 边缘胶带老化
- 影像粘连到盖玻璃上

### 修复思路

**切勿拆开**：
- 永远不要尝试分离两片玻璃
- 拆分意味着乳剂层可能毁损
- 受潮和霉变需交给专业人员
- 尽可能"原样"完成数字化

**数字化挑战**：
- 扫描时常出现牛顿环
- 多角度尝试可弱化干涉花纹
- 牛顿环严重时改用翻拍而非扫描
- 接受部分痕迹,这是物件历史的一部分

## 常见问题

### 怎么把老的玻璃底片数字化？

要把老的玻璃底片数字化,可将其乳剂面朝下放在带透射稿片附件的平板扫描仪上（以 2400—4800 DPI、48 位色深扫描）,或在灯箱上用带微距镜头的数码相机翻拍——这两种方法捕捉的都是负像,之后必须在数字端反相成正像。对玻璃底片而言,翻拍效果通常优于扫描。搬动时只捏边缘,佩戴棉手套,移动过程中始终全面托住,并尽快数字化,以便在物理劣化进一步加剧之前保留影像。

### 已经碎裂的玻璃底片还能数字化修复吗？

可以。已经碎裂的玻璃底片可以通过数字化方式修复——分别拍摄或扫描每一块碎片,再把所有碎片以独立图层导入图像编辑软件,凭借可见的细节进行精确对位,并用仿制图章和修复工具融合接缝。即便整片碎成几十块,只要碎片基本齐全,也能用数字方式拼合还原。复杂的破裂通常需要 3—8 小时。请把所有实物碎片一起放入保护性容器中。对于极为珍贵的玻璃片或破裂特别复杂的案例,可由专业修复机构先对碎片做物理稳定,再行数字化。

### 湿版底片和干版玻璃底片有什么区别？

湿版火棉胶底片（1851—1880 年代）需要在玻璃片仍处于湿润状态时完成涂布、增感、曝光与冲洗——必须就地在便携暗房中即拍即冲——所成影像极其锐利,但带有独特的不规则痕迹,乳剂层也非常脆弱。干版明胶底片（1871—1920 年代）出厂时已涂好,可长期存放后再用,乳剂层稳定得多,因而成为专业领域的标准。修复时,湿版往往呈现更多独特的瑕疵和乳剂不稳定问题,需要更轻柔地处理;干版整体保存得更好,但同样面临玻璃破裂和银影褪色的问题。

### 怎样安全保存玻璃感光板？

把玻璃感光板放进无酸的四面对折纸袋,在带档案级隔片的保护盒中竖立存放;湿度保持在 30—40%、温度 65—70°F、暗处保存,绝不平摞。每片都需独立护套（不允许玻璃片彼此直接接触）;竖立存放可避免重量压伤乳剂层;合格的环境控制能阻止进一步劣化。仅捏边缘,佩戴棉手套搬动,搬动过程中全面托住,并尽快数字化,把数字化当作防止意外破碎的"保险"。切勿使用 PVC、磁性相册或普通纸板。每年检查一次状态,留意新出现的损伤。

### 玻璃底片值得花钱请专业人士修复吗？

当玻璃底片承载着不可替代的历史影像、记录着重大事件或人物、出自知名摄影师之手,或具有家谱/法律意义时,就值得交给专业修复。专业修复每片需 $200—2000+,但能获得专家级稳定化、碎片再装配、霉菌清除以及合格的档案级保护。对于家庭收藏,DIY 数字化（设备投入 $0—500)加上数字端的修复（找专业人员每张 $50—200,自己来则只花时间）通常更划算。若遇到以下情况,请务必咨询专业人员：玻璃片碎裂、正在主动劣化、乳剂层翘起,或有博物馆级要求。
