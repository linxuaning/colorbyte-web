## Troubleshooting

### Common Issues

**"Already submitted"**
- Check spam for previous confirmation
- Try different email
- Contact directory support

**Rejection - "Not AI enough"**
- Emphasize neural network/deep learning technology
- Add technical details about restoration algorithms

**Rejection - "Too promotional"**
- Remove superlatives ("best", "amazing")
- Use factual language
- Focus on what it does, not why it's great

**No response after 2 weeks**
- Send polite follow-up email
- Resubmit with improved description
- Move to next directory
