## Part 6: Metadata Patterns Observed

### Strong Patterns (Maintain)

1. **Consistent title format:** `[Keyword] | [Benefit] - ArtImageHub` or similar
2. **Description length:** 150-160 characters (optimal)
3. **Keywords:** 5-10 relevant keywords per page
4. **OpenGraph images:** All use `/blog/before-after-examples.webp`
5. **Twitter cards:** summary_large_image for all pages
6. **Canonical URLs:** Present on all landing pages

### Recommended Template for New Pages

```typescript
export const metadata: Metadata = {
  title: "[Primary Keyword] | [Benefit] - ArtImageHub",
  description: "[160 char description with keyword, benefit, and CTA]",
  alternates: { canonical: "/page-slug" },
  keywords: ["keyword1", "keyword2", "..."],
  openGraph: {
    title: "[Title for social sharing]",
    description: "[Shorter social description]",
    type: "website",
    images: [{ url: "/blog/before-after-examples.webp", width: 1200, height: 630 }],
  },
  twitter: {
    card: "summary_large_image",
    title: "[Twitter title]",
    description: "[Twitter description]",
    images: ["/blog/before-after-examples.webp"],
  },
};
```

## Part 7: Schema Template for Missing Pages

### SoftwareApplication Schema (Copy for P0/P1 fixes)

```typescript
const softwareSchema = {
  "@context": "https://schema.org",
  "@type": "SoftwareApplication",
  name: "ArtImageHub [Feature Name]",
  applicationCategory: "MultimediaApplication",
  operatingSystem: "Web browser",
  offers: {
    "@type": "Offer",
    price: "0",
    priceCurrency: "USD",
    description: "3 free restorations, Pro Lifetime $29.9"
  },
  aggregateRating: {
    "@type": "AggregateRating",
    ratingValue: "4.8",
    ratingCount: "2847"
  }
};
```

### Script Tag Implementation

```tsx
<script
  type="application/ld+json"
```
