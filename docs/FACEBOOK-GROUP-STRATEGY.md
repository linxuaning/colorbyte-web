## Risk Mitigation

### Avoiding Removal

1. **Read group rules** before posting
2. **Don't mention tools** in first 2-3 posts
3. **Provide value** before any self-promotion
4. **Never post identical content** across groups
5. **Respect admin decisions** if removed

### Building Trust

1. **Engage genuinely** for 1-2 weeks before posting
2. **Help others** without expecting anything
3. **Share failures** as well as successes
4. **Be transparent** about what worked/didn't
